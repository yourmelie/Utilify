# Utilify · End of Life

**Official Support window: August 31, 2024 – April 15, 2026**    
> **Private Builds Support window: May 11, 2023 – April 15, 2026**

Utilify has reached the end of its active development. A combination of suboptimal internal architecture, limited maintenance bandwidth, and a genuine lack of motivation has led to this decision. While the script may still function in many scenarios, its current state is notably unstable and can trigger DOM crashes or unexpected page behavior.

As a small gesture of accountability for this abrupt sunset, a streamlined successor is already taking shape. **BlackStar** will focus exclusively on the essential utility layer, delivered with a cohesive visual design that adapts natively across all themes. Support for all three primary KoGaMa domains is planned.

## What Utilify Was

Utilify served as a comprehensive userscript suite for KoGaMa, bundling visual refinements, privacy controls, profile enhancements, and an extensible plugin system. It operated entirely within the page context with `@grant none` and required no elevated permissions.

## Core Capabilities (Abridged)

- **Visual Overhaul:** CSS corrections, glassmorphic DM panels, and accent‑color harmonization.
- **Background Effects Engine:** Syntax‑driven particle overlays (rain, snow, fireflies, etc.) triggered from profile descriptions.
- **Profile Bookmarks & Add Friend Button:** Persistent bookmarking with pinning, plus restored friend‑request controls.
- **Imgur Integration:** Paste screenshots directly into chat and hover previews for Imgur links.
- **Seamless Description Box:** Replaces truncated bios with scrollable, fully decoded text.
- **Faster Friends Panel:** Single‑request loading of up to 555 friends with search and mutual friend visibility.
- **Avatar Marketplace Finder:** Gradient‑linked avatar names with concurrent marketplace searches.
- **Plugin System:** External script loader with metadata extraction and storage persistence.
