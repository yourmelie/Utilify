# BlackStar `v1.7.3`

BlackStar is the practical, focused follow-up to Utilify. It drops the bloat and keeps what people actually used, wrapped in a native dark theme that works across all three KoGaMa domains (WWW, BR, FRIENDS).

Files are provided only on [GitHub](https://github.com/yourmelie/FormerlyUtilify).

**What it includes:**

- **Native dark theme** -- fully customizable through `:root` variable overrides
- **Compact profile** -- BetterLevel, SeamlessDesc, and Activity + Archive widgets
- **Feed manager** -- review profile posts and comments
- **Faster friends** -- fast-loading friends panel with search and mutual friend visibility
- **Restored invite friend button** -- back on profiles where it belongs
- **Playing type chip** 
- **Paste and dot obfuscation** -- re-enabled where it was stripped
- **Hide activity** -- always on
- **Spy friends' activity** -- always on

**Contributors:** Death Wolf., ReZa, Sorry, Raptor, TunA, Comenxo, Zpayer.

## Notice

**What led to Utilify being dropped?**
Honestly, everything. I have been on KoGaMa since February 2014 and gave it far more time and weight than was ever healthy. It was time to step back. The codebase situation did not help either, it had grown brittle and hard to touch without something else breaking. Maintenance kept slipping, motivation followed, and at some point replacing it felt more honest than pretending I would ever fix it. Letting it quietly rot was not something I wanted to do.

**Where are Utilify's older versions and other KoGaMa project files?**
Out of respect for my own peace and for the platform, none of that is accessible to third parties. Some research material may exist somewhere for my own reference, but none of it will resurface. If you have questions, I would genuinely prefer you let them go. That said, if something urgent or safety related comes up, you can reach me on Discord at <ins>simonvhs</ins>.

**Can we expect anything else from you going forward?**
No. I also make no promises about maintaining BlackStar. It exists, it works, and that is where my commitment ends.

## Utilify (Legacy)

**Support window: August 31, 2024 to April 15, 2026**

Utilify was a userscript suite for KoGaMa covering visual refinements, privacy controls, profile enhancements, and a plugin system. It ran entirely within the page context using `@grant none` with no elevated permissions. Development ended due to accumulated architectural debt and a loss of motivation to maintain it. It may still function in some cases but is no longer stable.

> No installation links/files are to be provided/shared.
